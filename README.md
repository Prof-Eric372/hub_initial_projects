# Python Terminal Projects Hub

A collection of small terminal-based projects built with Python, focused on consolidating the fundamentals of programming logic.

## Purpose

This repository was created as a practical study tool to reinforce core concepts in Python before moving on to more complex topics such as loops, functions, and external libraries. Every project here was written without the assistance of AI — the goal is to prove that the logic and structure were understood, not just copied.

The projects intentionally avoid external libraries. Everything runs on pure Python, using only what the language offers natively.

## Concepts covered

- Data types (`int`, `float`, `str`)
- Type conversion
- User input handling
- Arithmetic operations
- Conditional logic (`if`, `elif`, `else`)
- Formatted output with f-strings

## Projects

### 01 — IMC Calculator
The user enters their height in meters and weight in kilograms. The program calculates the Body Mass Index and returns the result alongside a classification ranging from underweight to obese.

### 02 — Discount Calculator
Simulates a store with three products on sale. The user selects an item and the program returns the final price after applying the corresponding discount percentage.

### 03 — Temperature Converter
The user enters a temperature in Celsius and the program converts it to both Fahrenheit and Kelvin, displaying all three values.
